# Retail Sales Performance Analysis

## Project Overview
This project focuses on analyzing retail sales data to identify key business insights related to product categories, regions, and seasonal trends.

## Business Objective
To understand sales performance across categories, regions, and time periods in order to support data-driven business decisions.

## Dataset
- Retail sales dataset
- transaction records
- Key fields: Order Date, Category, Region, Sales

## Tools & Technologies
- Python
- Pandas
- Matplotlib

## Analysis Performed
- Data cleaning and preprocessing
- Feature engineering (year, month, month name)
- Category-wise and region-wise sales analysis
- Yearly and monthly sales trend analysis
- Data visualization using bar and line charts

## Key Insights
- Technology is the highest revenue-generating category
- West region contributes the highest overall sales
- Sales peak in November and December, showing strong seasonality
- February records the lowest sales volume

## Outcome
The analysis highlights how sales data can be leveraged to identify revenue drivers, optimize regional strategies, and understand seasonal demand patterns.
